Definition
Kinematics the study of motion of objects using graphs and equations to describe that motion
Position where an object is at any particular time (relative to a refrence frame).
Displcaement the change in position of an object.
Distance total length traveled from initial position.
Vector any quantity with both magnitude and direction
Scalar Any quantity that has a magnitude but no direction
Elapsed Time difference between the ending time and the beginning time
Velocity the rate at which the position changes.
Instantaneous Speed the magnitude of instantaneous velocity.
Acceleration the rate at which velocity changes over time.
Free-Fall falling w/o air resistance or friction(the only force acting upon an object is gravity)
